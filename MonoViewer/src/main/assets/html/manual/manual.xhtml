アプリマニュアル

「モノちゃんびゅーわ」について

モノちゃんびゅーわ（以下、本アプリ）は千葉都市モノレール株式会社がwebで公開している時刻表データ、運行情報データを元に独自に解析を行い今どこに千葉モノレールが走行しているのかを表示する非公認アプリです。

本アプリの想定シーン

  1. モノちゃん号の写真を撮りたいのだけれど後どれくらい待てば良いのだろう？
  2. 駅の時刻表を確認するついでに新型車両かも確認したい！
  3. 今現在何台くらい走っているのかな？
  4. 今何処を走っているのだろう？

使い方

本日の運行状況

アプリを起動するとこの画面からはじまります。

千葉都市モノレールが公開している当日のモノちゃん号と新型車両の運行予定を検索できます。

※問い合わせには少し時間がかかります。

  • 車両名をクリックすると運行予定の列車運行時刻表が表示されます。
  • 画面下部の地図を開くをクリックすると運行中の千葉モノレールがGoogleMap上に表示されます。
  • オプションメニューからその他の画面も直接開くことが出来ます。

列車運行時刻表

千葉都市モノレールが事前に公開している、車両がどの駅を何時に発車するのかを検索します。

特定の車両を追跡する際に便利な表示となります。

走行予定の車両アイコンを左側に表示しているので一目でどの車両が予定されているか分かります。

アイコンの右に添えられているアイコンは上りと下りを表しています。

  • 駅名をクリックするとその駅時刻表を検索出来ます。
  • 左上のスピナーを開いて任意の車両の運行予定表を検索出来ます。
  • 右上のラジオボタンを選択することで休日・平日の運行予定表が検索出来ます。
  • オプションメニューからその他の画面も直接開くことが出来ます。

駅時刻表

駅で表示されている標準時刻表を検索できます。

左側に灰色で示しているのが時間

右側に続くのが分

分の上に表示されている注記は標準時刻表に準拠しています。

右側に表示されているのは車両情報で、HA1やKA1というのが運行情報のIDとなります。

右上に表示されているのがモノちゃん号及び新型車両の判定となります。

  • 時刻をクリックするとその時刻に通過する列車運行時刻表を検索出来ます。
  • 左上のスピナーを開いて任意の駅時刻表を検索出来ます。
  • 右上のラジオボタンを選択することで休日・平日/上り・下りの時刻表が検索出来ます。
  • オプションメニューからその他の画面も直接開くことが出来ます。

地図表示

現在走行中の千葉モノレールを疑似リアルタイムに閲覧出来ます。

列車アイコンは0.5s毎に予測現在位置を移動します。

アイコン右側に表示されている青と緑のアイコンは上りと下りを表しています。

  • 駅アイコンや列車アイコンをクリックするとバルーンが表示されます。
  • バルーンを更にクリックすると駅時刻表や列車運行時刻表を検索することが出来ます。
  • 右上のボタンを選択することで現在位置を検索することが出来ます。
  • GoogleMapに制限は設けていませんのでドラッグ等の動作で任意の地点を表示することが出来ます。

アイコンの説明

モノちゃん号を表すアイコン

アーバンフライ０型を表すアイコン

通常列車を表すアイコン

上り列車を表すアイコン

下り列車を表すアイコン
