モノちゃんびゅーわ（以下、本アプリ）は千葉都市モノレール株式会社がwebで公開している時刻表データ、運行情報データを元に独自に解析を行い時刻表データに変換したものを利用して情報を提供しております。

本アプリ作成にあたりミスがないよう細心の注意を払って作成しておりますが完璧さの提供は担保しておりません。これにより発生した損害に対する責任は一切負いかねますので、あらかじめご了承ください。

千葉都市モノレール株式会社による免責事項表記

運行情報は１週間に１回を目安として更新しておりますが、運行ダイヤが乱れたときなどは予定を変更して運行することもあります。このため本ページの情報と異なり、実際には運行してい ない、運転時刻が異っている、という可能性がありますが、これにより発生したお客様の損害に 対する責任は一切負いかねますので、 あらかじめご了承ください。
